Interop.setEventHandler

src/main/java/bt7s7k7/supervisory/compat/computercraft/InteropAPI.java:127

Overloads:

  • setEventHandler(peripheral: Interop.PeripheralReactiveDependency, handler: Function)

Sets an event handler on the specified peripheral. The handler will receive a event argument containing the String name of the event and an arguments argument containing an Array of the arguments composing the event. Two special events are emitted by this system: __connected__, when a peripheral is connected and __disconnected__ when a peripheral is disconnected.
