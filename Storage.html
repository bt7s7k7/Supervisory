Connects to an inventory, that will be scanned for contained items. The target can be either a side of this system or a name of a socket on the network. This function returns a Storage.StorageReactiveDependency that can watched for changes using the reactive function.

Storage.transfer

src/main/java/bt7s7k7/supervisory/storage/StorageAPI.java:70

Overloads:

  • transfer(items: Array, targets: Array)

Transfers all item stacks in the items argument into the targets inventories. The items array should contain Storage.StackReport instances for all items that should be transferred. The targets array should contain Storage.StorageReport instances for all inventories that should be transferred into. If there is not enough space to transfer items, only a partial transfer is performed. The amount of transferred items is returned.
